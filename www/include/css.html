Dispor elementos em linha desconsiderando espaços em branco

Neste texto irei apresentar uma maneira simples e eficaz em dispor elementos em bloco, um lado de outro, desconsiderando os espaços em branco que forçam os elementos a quebrarem de linha.

E para esta situação não queremos usar "float", nem "flex box" e "box sizing" ... hmmm ... não vai ajudar agora.

Citando como exemplo a estrutura HTML e estilos abaixo ...

<div id="pai">
    <div class="filho filho-1"></div>
    <div class="filho filho-2"></div>
</div>
    #pai .filho {
    display: inline-block;
    width: 50%;
}

... Sabemos que os elementos ".filho" se comportarão em linha e block e terão metade da largura do elemento "#pai". Mas eles não ficarão lado a lado. Isto ocorre por causa dos espaços em branco presentes no documento HTML. Para resolver isto insira o estilo citado abaixo:

#pai {
    font-size: 0;
}

Definindo o "font-size" para "0" nós "removemos" o espaço em branco presente no documento. O único incoveniente é que os elementos ".filho" herdarão este estilo, então o "font-size" deverá ser definido para eles, resultando num código similar a este:

#pai {
    font-size: 0;
}
#pai .filho {
    display: inline-block;
    width: 50%;
    font-size: 1.2rem;
}

É uma técnica bem simples mas que lhe auxiliará muito em seus projetos!
