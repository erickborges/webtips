HTML Clique!

Mostrarei neste artigo como usar eventos de clique com HTML. Com essa técnica você poderá criar accordions, como será mostrado, sem uso de JS

Quando queremos mostrar conteúdo em um website em forma de abas, por exemplo, logo pensamos em usar Javascript, jQuery ou outra biblioteca js para facilitar nosso trabalho. Porém, em algumas situações, complicamos a implementação ao invés de simplificá-la.

Ao usarmos um componente de abas, ou até mesmo de accordion, feito apenas com HTML e CSS teremos certeza que irá funcionar em navegadores com javascript desabilitado, navegadores mobile - sendo que bibliotecas javascript podem não funcionar corretamente neles - e web app's.

Uma vez me disseram que não há cliques em HTML. Mas e o radio button? E o checkbox? Quando são clicados podem ser associados ao psedo-elemento :checked do CSS, como mostrado abaixo:

<div id="pai">
    <h3>Radio</h3>
    <input type="radio" name="clickRadio" id="click1" />
    <label for="click1">Click 1</label>
    <input type="radio" name="clickRadio" id="click2" />
    <label for="click2">Click 2</label>
</div>

<div id="pai">
    <h3>Checkbox</h3>
    <input type="checkbox" id="click3" />
    <label for="click3">Click 3</label>
    <input type="checkbox" id="click4" />
    <label for="click4">Click 4</label>
</div>

Como podem ver, o importante desta técnica é usar o atributo for no label possuindo o mesmo valor do atributo id do input. Assim, o click no label marca o input correspondente.

E para criar o componente de abas veja a estrutura HTML e CSS abaixo:

<div id="accordionGroup">
    <div class="accordion">
        <input type="radio" name="clickAccordion" id="accordionClick1" />
        <h3><label for="accordionClick1">Accordion Click 1</label></h3>
        <div class="accordion-content">Accordion Content 1</div>
    </div>
</div>

input[type="radio"] {
    display: none;
}
h3 {
    color: red;
}
input:checked + h3 {
    color: green;
}
.accordion-content {
    display: none;
}
input:checked ~ .accordion-content {
    display: block;
}

Colocar display: none; no input não afeta o funcionamento da técnica.

Então define-se os estilos padrões dos títulos, labels e conteúdo. Através dos operadores CSS + e ~ vinculado ao input clicado define-se os estilos para os elementos que ficarão em evidência.

O CSS do exemplo apenas mostra/oculta o conteúdo do accordion. Uma boa dica é usar animações e transições do CSS 3 para você criar efeitos interessantes.
